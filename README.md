# Library Management System

## Overview
The Library Management System is a comprehensive software solution designed to manage the day-to-day operations of a library. It provides functionalities for managing books, members, borrowing records, and events.

## Features
- **Book Management**: Add, update, and delete books from the library's catalog.
- **Member Management**: Register new members and manage existing member information.
- **Borrowing Records**: Track the borrowing history of books by members.
- **Event Management**: Organize and manage library events, including registrations.

## Getting Started
### Prerequisites
- .NET Framework
- Entity Framework
- SQL Server

### Installation
1. Clone the repository:
```bash
git clone https://github.com/codelotusr/LibraryManagementSystem.git
```
2. Open the solution in Visual Studio.
3. Restore NuGet packages.
4. Build and run the application.

## Usage
- Navigate through the tabs to access different functionalities.
- Use the forms provided to add or update information in the system.
- Select items from the list views to view or modify details.
